запуск проекта
- установка зависимостей
```bash
npm install --legacy-peer-deps
```
- запуск
```bash
npm start
```
запуск прокси(находясь в папке proxy) для доступа к данным с localhost 

- установка зависимостей прокси 
```bash
npm install
```
- запуск прокси
```bash
npm run start:proxy
```

запускался и тестировался на macOS
